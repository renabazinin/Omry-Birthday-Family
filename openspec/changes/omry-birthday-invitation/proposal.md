## Why

Omry is turning 4 and needs a fun, eye-catching digital birthday party invitation to send to friends and their parents. The party is on Friday, April 10th 2026. A web-based invitation is easy to share via WhatsApp/messaging and looks more engaging than a plain text message. The audience is Hebrew-speaking families, so all visible content must be in Hebrew with full RTL support.

## What Changes

- Create a single-page animated birthday invitation website
- All user-facing text in Hebrew, fully RTL layout
- Fun kid-friendly animations (balloons, confetti, stars, etc.)
- Mobile-first responsive design (parents will open on phones)
- Simple modular codebase: separate HTML, CSS, and JS files
- Static site — no build tools, no frameworks, no dependencies

## Capabilities

### New Capabilities
- `invitation-page`: Main HTML page with party details (who, when, where), Hebrew RTL content, and responsive layout
- `animations`: CSS and JS animations — balloons, confetti, bounce effects — to make the invitation playful and engaging for kids
- `responsive-layout`: Mobile-first responsive design ensuring the invitation looks great on phones, tablets, and desktops
